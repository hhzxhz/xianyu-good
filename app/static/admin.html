网页通道 · 从 cURL 提取登录

在 Chrome 开发者工具中对任意 goofish.com / h5api.m.goofish.com 请求右键 复制 → 复制为 cURL，将整段粘贴到下方。点击「解析并应用」后写入服务端内存，优先于 .env 中的 WEB_STORAGE_STATE_PATH / WEB_CURL_COOKIE_FILE。 正在运行的网页抢购任务会在下一轮循环自动关闭并重启浏览器以应用新 Cookie。

请勿在终端直接执行该 cURL 去请求 h5api（mtop… 的 sign/t 会过期，且易返回 404/风控）。此处只提取 Cookie 给内置浏览器用，不重放该 HTTP 请求。
